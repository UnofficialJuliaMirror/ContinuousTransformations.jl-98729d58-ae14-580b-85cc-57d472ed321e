    ContinuousTransformations.TransformLogLikelihood — Type.
    TransformLogLikelihood(ℓ, transformations::Union{Tuple, TransformationTuple})
TransformLogLikelihood(ℓ, transformations...)

    Return a callable that

    1. transforms its vector argument using a transformation tuple (or a tuple of

    transformations, converted as required) to a tuple of values,

    1. calls ℓ with these, which should return a scalar,

    2. returns the result above corrected by the log Jacobians.

    Useful when ℓ is a log-likelihood function with a restricted domain, and transformations is used to trasform to this domain from $ℝ^n$.

    See also get_transformation.
